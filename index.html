Grupo de Trabalho de Tecnologia da Informação do Partido Pirata - GTI

Bem vinda(o) ao GTI Pirata

O Grupo de Trabalho de Tecnologia da Informação do Partido Pirata (GTI) se organiza em diversas ferramantas. Para começar a participar, escolha a que considerar mais prática.

Mais informações?

Todas as informações estão consolidadas em uma wiki. Em caso de dúvidas, o melhor caminho é enviar e-mail para gti@partidopirata.org. Para criptografia, veja a página de PGP na wiki.

Loomio

Loomio é uma ferramenta para facilitar a tomada de decisão de forma colaborativa. É uma ferramenta utilizada pelo Partido Pirata e existe um sub grupo no Loomio para o GTI.

Todas as decisões do GTI deve(ria)m ser registradas neste sub grupo do Loomio.

gti.partidopirata.org/loomio »

Github

Github é um serviço comercial de software proprietário que utilizamos para facilitar a organização de códigos fonte. Também utilizamos para organização e comunicação interna de forma pública.

gti.partidopirata.org/github »

Telegram

Telegram é um cliente de mensagem instantânea de software que usa um serviço de código fechado. As(os) piratas utilizam muito o telegram para bate papo.

canal do gti »

grupo aberto do gti »

Ikiwiki

A wiki serve para o Partido e não somente para o GTI, e tem por finalidade armazenar informações e documentar o que é necessário.

gti.partidopirata.org/wiki »

Taiga

Taiga é uma ferramenta para organização de projetos que utilizamos para dividir tarefas e trabalho.

gti.partidopirata.org/taiga »

NotABug

Notabug é um site que usa Gogs, um clone do github em software livre e é utilizado principalmente por pessoas que não querem ou não podem usar github. A maioria dos sites do Partido Pirata é atualizado diretamente a partir dos repositórios do notabug.

gti.partidopirata.org/notabug »
